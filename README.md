Volunteer Teaching Platform

Description:
This web application connects volunteer teachers with underprivileged students, providing a platform for learning, mentorship, and donations. The system includes modules for students, teachers, donors, and admins, enabling seamless management of study materials, quizzes, study sessions, feedback, donations, and complaints.

Key Features:

Student Module: Registration/login, access study materials, attempt quizzes, join online sessions, submit feedback and complaints.

Teacher Module: Registration/login, upload study materials, create quizzes, conduct online sessions, monitor student progress.

Donation Module: Donor registration/login, make donations, view donation history, support targeted campaigns.

Admin Module: Manage users, grades, subjects, announcements, and complaints.

Technologies Used:

MERN Stack: MongoDB, Express.js, React.js, Node.js

Tools: GitHub (version control), Visual Studio Code, Microsoft Teams, WhatsApp

Online Sessions: Zoom integration for live classes

System Overview:
The system architecture integrates all modules to facilitate efficient learning management, communication, and monitoring. Students gain access to learning resources and sessions, teachers manage content and track progress, donors contribute to support education, and admins oversee system operations.
